# 大模型价格展示

> 每次对比各家大模型的的价格都很犯愁，要么去各家网站上看，要么就只能去各种整个站点看，都不够直观。也没有统一的数据源，所以这个站点就诞生了，目的就是可能没有任何附加条件的情况下，看到所有（尽可能）—的模型的价格

一个用于对比和展示大模型价格信息的现代化 Web 应用。

## 🚀 功能特性

- **Agent 自驱动**：Agent 自我驱动更新内容
- **多币种显示**：支持人民币/美元切换与汇率换算，支持显示原声计算货币
- **分层定价**：支持按上下文长度的阶梯价格
